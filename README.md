# Predição de Tempo de Internação utilizando Regressão Linear (OLS)

Implementei esse trabalho para a disciplina de Introdução à Machine Learning. Nessa atividade eu precisei aplicar a fórmula fechada de regressão linear (OLS), utilizando apenas **NumPy**, em um **dataset real** de minha escolha. Utilizei a solução do método dos mínimos quadrados:

$$\hat{\theta} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{y}$$

onde $\mathbf{X}$ é a matriz de features (com a coluna de 1s para o intercept) e $\mathbf{y}$ é o vetor de targets.
